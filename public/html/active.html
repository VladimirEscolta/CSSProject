:active

Описание

Псевдокласс :active определяет стиль активного элемента. Это такое состояние элемента, которое происходит между щелчком и отпусканием клавиши мыши. В основном применяется к ссылкам и кнопкам, но ими не ограничен.

Синтаксис

Селектор:active { ... }

Пример

.pr1 {
                margin: 20px;
                padding: 5px;
                width: 300px;
                height: 100px;
                text-align: center;
                background: orange;
                border: 2px solid black;
                border-radius: 10px;
            }
            
            .pr1:active {
                box-shadow: 0 0 30px 20px violet;
            }
Нажми на меня и удерживай кнопку нажатой!!!
