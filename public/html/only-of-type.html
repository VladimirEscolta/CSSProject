:only-of-type

Описание

Псевдокласс :only-of-type применяется к дочернему элементу указанного типа, только если он единственный у родителя.

Синтаксис

Селектор:only-of-type { ... }

Пример

.pr1 ol:only-of-type {
                font-size: 25px;
                font-style: italic;
                color: rgb(255, 145, 0);
            }
