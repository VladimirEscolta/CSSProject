order

Описание

Определяет порядок вывода флексов внутри флекс-контейнера. Элементы располагаются согласно значениям свойства order от меньшего к большему. При равных значениях order элементы выводятся в том порядке, в каком они появляются в исходном коде. В качестве значения принимается любое целое число, включая отрицательные числа и ноль.

Синтаксис

order: <число>

Пример

.flex {
                list-style: none; 
                display: flex; 
                flex-flow: row wrap;
            }
            
            .order {
                background: #489d75;
                color: white;
                padding: 20px;
                margin: 5px; 
                font-size: 2em;
            }
            
            .pr1 {order: 5;}
            .pr2 {order: 3;}
            .pr3 {order: 4;}
            .pr4 {order: 1;}
            .pr5 {order: 2;}
