align-self

Описание

Свойство align-self выравнивает флекс-элементы текущей строки, переписывая значение align-items.

Синтаксис

align-self: auto | flex-start | flex-end | center | baseline | stretch

Пример

auto 

                .primer {
                    display: flex;
                  }

                .block2 {
                    align-self: auto;
                  }
