clip

Описание

Свойство clip определяет область позиционированного элемента, в которой будет показано его содержимое. Всё, что не помещается в эту область, будет обрезано и становится невидимым. На данный момент единственная доступная форма области — прямоугольник. clip работает только для позиционированных элементов, у которых свойство position задано как absolute или fixed.

Данное свойство устарело, вместо него используйте clip-path.

Синтаксис

clip: rect(Y1, X2, Y2, X1) | auto

Пример

                .clip {
                    animation: clip 3s infinite;
                    position: absolute;
                    width: 200px;
                    color: white;
                    background: rgb(255, 145, 0);
                    padding: 10px;
                }
                
                @keyframes clip {
                    from { clip: rect(auto, auto, 0, auto); }   
                    to { clip: rect(auto, auto, 200px, auto); }  
                }
