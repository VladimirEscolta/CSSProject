zoom

Описание

Изменяет масштаб объекта согласно заданному значению. Это нестандартное свойство, вместо него используйте transform.

Синтаксис

zoom: <число> | <проценты> | normal

Пример

.zoom {
                margin: 15px;
                padding: 5px;
                width: 300px;
                border: 2px solid sandybrown;
            }
            
            .zoom:hover {
                zoom: 200%;
            }
