@media

Описание

Правило @media позволяет указать тип носителя, для которого будет применяться указанный стиль. В качестве типов выступают различные устройства, например, принтер, монитор и др.

Синтаксис

@media <тип носителя> {<стилевые правила>}

Пример

@media screen { 
                body {
                 font-family: Arial, Verdana, sans-serif; 
                 font-size: 0.9em; 
                 color: #000080; 
                }
Типы носителей и их описание
Тип Описание
all Все типы. Это значение используется по умолчанию.
print Принтеры и другие печатающие устройства.
screen Экран монитора.
speech Речевые синтезаторы, а также программы для воспроизведения текста вслух. Сюда, например, можно отнести речевые браузеры.

После ключевого слова @media идёт один или несколько типов носителя; если их больше одного, то они разделяются между собой запятой. После чего следуют обязательные фигурные скобки, внутри которых пишутся стилевые правила.
