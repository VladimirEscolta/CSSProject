:root

Описание

Псевдокласс :root определяет корневой элемент документа. В HTML этот селектор всегда соответствует элементу <html>, но в стилях приоритет :root выше, чем у селектора html.

Синтаксис

:root { ... }

Пример

:root {
                background-color: pink;
                font-style: italic;
            }
