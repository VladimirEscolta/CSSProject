box-decoration-break

Описание

Указывает, как оформлять фрагмент текста при его переносе на другую строку. Работает только для следующих свойств: background, border, border-image, box-shadow, clip-path, margin и padding.

slice
Каждый фрагмент разрезается в месте переноса и никакие эффекты оформления на месте «разреза» не отображаются. В частности, граница не показывается, тень не выводится, радиус скругления работает толькодля всего блока целиком.
clone
Каждый фрагмент оформляется независимо. Ко всем фрагментам отдельно применяется рамка, тень, фон и др.

Синтаксис

box-decoration-break: slice | clone

Пример

                .primer1 {
                    margin: 10px;
                    width: 250px;
                    padding: 5px;
                    line-height: 2;
                    border: 2px double red;
                    background-color: sandybrown;
                }
                
                #break {
                    text-decoration: none;
                    border: 3px solid blue;
                    box-decoration-break: clone;
                }
