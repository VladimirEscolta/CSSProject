:link

Описание

Псевдокласс :link применяется к ссылкам, которые ещё не посещались пользователем, и задаёт для них стилевое оформление.

Синтаксис

Селектор:link { ... }

Пример

.link {
                font-size: 1.5em;
            }
            
            .link:link {
                color: blue;
            }
            
            .link:active {
                color: red;
            }
