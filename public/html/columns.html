columns

Описание

Универсальное свойство, которое позволяет одновременно задать ширину и количество колонок многоколоночного текста.

Синтаксис

columns: column-width || column-count

Пример

columns 

                .columns {
                    padding: 5px;
                    columns: 3 60px;
                }

Определяет, как содержимое должно распределяться внутри колонок.

Колонки заполняются последовательно.
balance
Содержимое равномерно распределяется между колонок. При печати это правило относится только к последней странице.
balance-all
Содержимое равномерно распределяется между колонок. При печати это правило относится ко всем страницам.

Определяет, как должен отображаться элемент в многоколоночном тексте — занимать ширину всех колонок или только одну из них. Это свойство обычно применяют для заголовков текста или изображений, которые не должны разбиваться на колонки.

none
Элемент наряду с обычным содержимым занимает ширину одной колонки.
all
Элемент занимает все колонки.
