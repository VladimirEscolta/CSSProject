width

Описание

Устанавливает ширину содержимого элемента. По умолчанию ширина зависит от типа элемента: блочные занимают всю доступную ширину; ширина строчно-блочных равна ширине их содержимому. Свойство width позволяет явно задать желаемую ширину элемента, несмотря на его исходное поведение. Если для элемента свойство box-sizing задано как border-box, то width определяет ширину блока.

Синтаксис

width: <размер> | <проценты> | auto

Пример

.width {
                margin: 5px;
                padding: 5px;
                background: orange;
                width: 200px;
            }
