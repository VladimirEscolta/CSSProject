height

Описание

Устанавливает высоту содержимого элемента. По умолчанию высота определяется автоматически, исходя из содержимого элемента, но если задать фиксированное значение высоты, то она будет установлена, несмотря на объём содержимого. Если содержимое элемента превышает указанное значение высоты, то высота элемента останется неизменной, а содержимое будет отображаться поверх него. Из-за этой особенности может получиться наложение содержимого элементов друг на друга, когда элементы в коде HTML идут последовательно. Чтобы этого не произошло, добавьте свойство overflow со значением auto к стилю элемента. Если для элемента свойство box-sizing задано как border-box, то height определяет высоту блока.

Синтаксис

float: left | right | none

Пример

  
                .height {
                    border: 2px dotted red;
                    height: 100px;
                }
