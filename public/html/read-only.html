:read-only

Описание

Применяется к полям формы, у которых задан атрибут readonly. Такое поле не может быть модифицировано, при этом оно получает фокус, а данные в таком поле можно выделить и скопировать.

Синтаксис

input:read-only { ... }

Пример

.pr1 input:read-only {
                text-align: center;
                opacity: 0.5;
                border: 2px solid red;
                border-radius: 5px;
            }
