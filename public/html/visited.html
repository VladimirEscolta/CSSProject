:visited

Описание

Псевдокласс :visited применяется к ссылкам, уже посещённым пользователем, и задаёт для них стилевое оформление.

Синтаксис

a:visited { ... }

Пример

.link {
                font-size: 1.5em;
            }
            
            .link:link {
                color: blue;
            }
            
            .link:active {
                color: red;
            }
            
            .link:visited {
                color: gray;
            }
