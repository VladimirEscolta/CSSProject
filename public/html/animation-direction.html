animation-direction

Описание

Устанавливает направление движения анимации.

Синтаксис

animation-direction: normal | alternate | reverse | alternate-reverse

Пример

normal 

                .primer1 {
                    background: #fc0;
                    margin: 10px;
                    width: 50px;
                    height: 50px;
                    border-radius: 50%;
                    position: relative;
                    animation: text 5s normal infinite;
                }
                
                @keyframes text {
                    from { left: 0; }
                    to { left: 100%; transform: translateX(-80px); }
                }

Анимация идёт с самого начала, после завершения цикла возвращается к исходному состоянию.

alternate 

                .primer2 {
                    background: #fc0;
                    margin: 10px;
                    width: 50px;
                    height: 50px;
                    border-radius: 50%;
                    position: relative;
                    animation: text 5s alternate infinite;
                }

                @keyframes text {
                    from { left: 0; }
                    to { left: 100%; transform: translateX(-80px); }
                }

Анимация идёт с начала до конца, затем плавно возвращается в исходное положение.

reverse 

                .primer3 {
                    background: #fc0;
                    margin: 10px;
                    width: 50px;
                    height: 50px;
                    border-radius: 50%;
                    position: relative;
                    animation: text 5s reverse infinite;
                }
                
                @keyframes text {
                    from { left: 0; }
                    to { left: 100%; transform: translateX(-80px); }
                }

Анимация идёт с конца цикла, после его завершения возвращается к исходному состоянию.

alternate-reverse 

                .primer4 {
                    background: #fc0;
                    margin: 10px;
                    width: 50px;
                    height: 50px;
                    border-radius: 50%;
                    position: relative;
                    animation: text 5s alternate-reverse infinite;
                }
                
                @keyframes text {
                    from { left: 0; }
                    to { left: 100%; transform: translateX(-80px); }
                }

Анимация идёт с конца до начала, затем плавно возвращается в исходное положение.
