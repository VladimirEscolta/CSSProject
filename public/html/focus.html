:focus

Описание

Псевдокласс :focus определяет стиль для элемента, получающего фокус. Например, им может быть текстовое поле формы, в которое устанавливается курсор.

Синтаксис

Селектор:focus { ... }

Пример

.pr1, .forms, .one {
                margin: 5px;
            }
            
            .labels {
                width: 100px;
                float: left;
                text-align: right;
                padding-right: 10px;
            }
            
            .getfocus:focus {
                background: orange;
                box-shadow: 0 0 20px 2px red;
            }
