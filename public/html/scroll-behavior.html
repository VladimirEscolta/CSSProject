scroll-behavior

Описание

Определяет поведение прокрутки внутри элемента — плавная прокрутка или мгновенный переход.

auto
Переход к элементу внутри блока с прокруткой происходит мгновенно.
smooth
Прокрутка происходит плавно. Время и функция перехода устанавливаются непосредственно браузером.

Синтаксис

scroll-behavior: auto | smooth

Пример

.scroll {
                margin: 5px;
                padding: 3px;
                width: 100%;
                height: 100%;
                scroll-behavior: smooth;
            }
            
            #block1, #block2, #block3 {
                margin: 5px 5px 25px;
                padding: 3px;
                width: 80%;
                height: 500px;
                overflow: hidden;
                border: 1px solid black;
            }
            
            #block1 {background: yellow;}
            #block2 {background: green;}
            #block3 {background: violet;}
            
            nav {
                display: flex;
                place-content: center center;
                flex-wrap: wrap;
                width: 80%;
                margin: 10px;
                padding: 10px;
                border: 2px solid black;
                border-radius: 10px;
            }
            
            nav a { 
                margin: 10px;
                padding: 10px;
            }
