:first-child

Описание

Псевдокласс :first-child задаёт стиль первого элемента в группе братских элементов (имеющих одного родителя).

Синтаксис

Селектор:first-child { ... }

Пример

.pr1 p:first-child {
                font-size: 25px;
                font-style: italic;
                color: red;
            }
