:not

Описание

Псевдокласс :not задаёт правила стилей для элементов, которые не содержат указанный селектор. В качестве селектора могут указываться единичные псевдоклассы, теги, идентификаторы, классы и селекторы атрибутов. Нельзя использовать псевдокласс :not (конструкция :not(:not(...)) запрещена) и псевдоэлементы.

Синтаксис

Селектор:not(<Селектор>) { ... }

Пример

span {
                font-size: 1.5em;
                color: red;
            }
            
            span:not([class="red"]) {
                color: blue;
                text-shadow: 0 0 20px blue;
            }
