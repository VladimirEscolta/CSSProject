bottom

Описание

Устанавливает положение нижнего края содержимого элемента без учёта толщины рамок и отступов. Отсчёт координат зависит от свойства position, оно обычно принимает значение relative (относительное положение) или absolute (абсолютное положение).

При относительном позиционировании элемента, отсчёт ведётся от нижнего края исходного положения элемента, а при абсолютном — относительно нижнего края окна браузера. Если для родительского элемента задано position: relative, то абсолютное позиционирование дочерних элементов определяет их положение от нижнего края родителя.

Синтаксис

bottom: <размер> | <проценты> | auto

Пример

                .primer {
                    display: flex;
                    align-items: flex-end;
                    padding-left: 10px;
                    width: 85%;
                    border: 2px solid rgb(127, 197, 255);
                }
                
                .primer1 {
                    border: 5px double red;
                    background-color: sandybrown;
                    position: relative;
                    bottom: 30px;
                }
