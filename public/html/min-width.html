min-width

Описание

Устанавливает минимальную ширину элемента. Если окно браузера по ширине становится меньше заданной минимальной ширины элемента, то ширина элемента остается неизменной, а в окне появляется горизонтальная полоса прокрутки. Значение ширины элемента будет вычисляться в зависимости от установленных значений свойств width, max-width и min-width.

Синтаксис

min-width: <размер> | <проценты>

Пример

.min {
                border: 2px solid red;
                min-width: 400px;
            }
