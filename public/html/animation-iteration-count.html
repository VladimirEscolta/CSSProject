animation-iteration-count

Описание

Устанавливает направление движения анимации.

Синтаксис

animation-iteration-count: infinite | <число>

Пример

infinite 

                .primer1 {
                    background: #fc0;
                    margin: 10px;
                    width: 50px;
                    height: 50px;
                    border-radius: 50%;
                    position: relative;
                    animation: text 5s normal infinite;
                }
                
                @keyframes text {
                    from { left: 0; }
                    to { left: 100%; transform: translateX(-80px); }
                }

Любые комбинации значений, разделяемых между собой пробелом, определяющих параметры анимации.

<число> 

                .primer2 {
                    background: #fc0;
                    margin: 10px;
                    width: 50px;
                    height: 50px;
                    border-radius: 50%;
                    position: relative;
                    animation: text 5s alternate infinite;
                }

                @keyframes text {
                    from { left: 0; }
                    to { left: 100%; transform: translateX(-80px); }
                }

Анимация идёт с начала до конца, затем плавно возвращается в исходное положение.
