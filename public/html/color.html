color

Описание

Определяет цвет текста элемента.

Синтаксис

color: <цвет> | transparent

Пример

                .primer1 {
                    margin: 20px;
                    padding: 10px;
                    color: green;   
                }
                .primer1::first-letter {
                    color: red;
                    font-size: 40px;
                    font-family:'Times New Roman', Times, serif;  
                }
