:first-of-type

Описание

Псевдокласс :first-of-type задаёт стиль первого элемента определённого типа в группе братских элементов (имеющих одного родителя).

Синтаксис

Селектор:first-of-type { ... }

Пример

.pr1 p:first-of-type {
                font-size: 25px;
                font-style: bold;
                color: blue;
                text-shadow: 0 0 20px red;
            }
